# gbrn

go batch rename

## build

```
$ make
```

## e.g.

```
$ls
1.txt
2.txt
3.txt

$ make
$ ./gbrn

# open vim
- 1.txt
- 2.txt
- 3.txt
+ 01.txt
+ 02.txt
+ 03.txt

$ls
01.txt
02.txt
03.txt
```
